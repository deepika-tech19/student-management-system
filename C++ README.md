# Student Management System (C++)

This is a simple Student Management System developed using C++.

## Features
- Add Student
- Display Students
- Search Student by ID
- Delete Student

## Concepts Used
- Arrays
- Structures
- Loops
- Conditional Statements

## How to Run
1. Compile the code using any C++ compiler
2. Run the program
3. Follow the menu options

## Project Description
This project is a menu-driven application that allows users to manage student records. It performs basic CRUD operations (Create, Read, Update, Delete) using arrays.
